# Pac-Man 吃豆人游戏

基于原生 JavaScript + Canvas 实现的经典吃豆人游戏，完整实现幽灵 AI 系统和 A* 路径规划算法。

## 核心亮点

### 1. A* 路径规划算法
- 用于幽灵追踪玩家的最短路径计算
- 启发式函数：曼哈顿距离
- 性能优化：限制寻路范围 + 路径缓存

### 2. 四种不同策略的幽灵 AI

| 幽灵 | 类型 | 策略描述 |
|------|------|----------|
| 红色 | 激进追击型 | 始终追踪吃豆人当前位置，速度随游戏进度增加 |
| 粉色 | 预判拦截型 | 瞄准吃豆人前方4个单位的预判位置 |
| 青色 | 协同包围型 | 结合红色幽灵位置计算拦截点，形成包抄 |
| 橙色 | 警戒巡逻型 | 距离>8格时追击，≤8格时转为巡逻模式 |

### 3. 状态机系统
- 追击状态（Chase）：主动追踪玩家
- 分散状态（Scatter）：前往各自地图角落
- 恐惧状态（Frightened）：吃豆人吃超级豆子后，幽灵逃跑

### 4. 特色机制
- 普通能量豆：增加分数，吃完所有豆子获胜
- 特殊能量豆：位于迷宫的特殊节点，激活后使吃豆人获得临时高级权限，可重写幽灵协议

## 如何运行

1. 下载本仓库所有文件
2. 双击 `index.html` 用浏览器打开
3. 或者使用本地服务器：`python -m http.server 8000`

## 技术栈

- JavaScript (ES6)
- HTML5 Canvas
- A* 路径规划算法
